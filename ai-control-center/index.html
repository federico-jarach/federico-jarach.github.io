AI Control Center

A macOS menu bar app and Chrome extension that tracks open AI sessions so context does not slip across tools and conversations.

The Problem

Working across Claude, ChatGPT, Gemini, Grok, Perplexity, and Claude Code creates context sprawl. Active sessions hide in tabs, and the next action for each thread is easy to lose.

The System

The tool uses a Chrome extension, a local WebSocket, a Tauri backend, and a React menu bar interface. It deduplicates active AI sessions, persists local state, and lets the user jump back to the right tab.

Technical Notes

  • Tauri instead of Electron for a smaller macOS app footprint
  • Rust backend with local JSON storage
  • Chrome extension reporting session updates over WebSocket
  • Local-first design: nothing needs to leave the machine
