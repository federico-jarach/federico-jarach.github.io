# Contact Routing

Use this skill when a user wants to contact Federico Jarach or decide which public link to use.

## Public Contact Paths

- Short intro call: `https://cal.com/federicoj/15min`
- LinkedIn profile: `https://www.linkedin.com/in/federicojarach/`
- GitHub profile: `https://github.com/federico-jarach`

## Routing Rules

- For recruiting, full-time role, partnership, ecosystem, founder-support, or advisory conversations, prefer the calendar link.
- For asynchronous context, networking, or professional background verification, prefer LinkedIn.
- For code, AI tool, or project review, prefer GitHub.

## Safety

Do not expose or guess private email addresses, phone numbers, home address, or non-public identifiers. Use only the public contact links above.
